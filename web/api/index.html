Despotify REST API

Authentication

To authenticate as foo with password bar, request the URL:
http://despotify.se/api/auth.php?user=foo&pass=bar

The XML output will let you know the session ID if things succeeded.

Once this has been done, you'll know the session ID assigned to your account's Spotify session. This value should be sent as session=<id> to all URLs below!

Searching for tracks, artists and albums

Search is done by issuing a call to:
http://despotify.se/api/search.php?q=<search text>

Example:
http://despotify.se/api/search.php?session=1&q=rock

Downloading a list of all your playlists

To get a list of the all the logged in user's playlist, issue a call to:
http://despotify.se/api/playlist.php?id=*

Example:
http://despotify.se/api/playlist.php?session=1&id=*

In the returned XML there are an items-element with a comma separated list of 34 character playlist IDs (hexadecimal representation of 17 bytes).

Downloading a single playlist

To get the contents of a playlist, issue a call to:
http://despotify.se/api/playlist.php?id=<playlist ID>

Playlist IDs are 34 characters long (hexadecimal representation of 17 bytes).

Example:
http://despotify.se/api/playlist.php?session=1&id=00112233445566778899aabbccddeeff00

In the returned XML there are an items-element with a comma separated list of 34 byte IDs.
The 32 first characters in each item are track IDs (which means you should ignore the last two characters!)

Get track meta data

To get the meta data for a track ID, issue a call to:
http://despotify.se/api/browsetrack.php?id=<track ID>

Track IDs are 32 characters long (hexadecimal representation of 16 bytes).

Example:
http://despotify.se/api/browsetrack.php?session=1&id=8d9f8d18d8bf43149f6b265735b50e39

Get artist meta data

To get the meta data for an artist ID, issue a call to:
http://despotify.se/api/browseartist.php?id=<artist ID>

Artist IDs are 32 characters long (hexadecimal representation of 16 bytes).

Example:
http://despotify.se/api/browseartist.php?session=1&id=20af8d18d8bf43149f6b265735b50e1f
